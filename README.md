# Shopify Uploader V4 🚀

Automated Shopify product uploader built for real-world ecommerce automation workflows.

This system allows you to upload products from structured data (CSV/JSON) directly into a Shopify store — including variants, pricing, inventory, and images.

---

## ⚙️ Features

- Bulk product upload
- Variant handling (price, SKU, inventory)
- Image upload support
- Shopify GraphQL API integration
- Designed for automation pipelines

---

## 📦 Sample Input

```json
{
  "title": "Kitchen Organizer Rack",
  "price": "9.99",
  "images": ["https://example.com/image.jpg"]
